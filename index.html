Getting Started

In this blog I am going to tell you how to protect any folder in windows without using any additional softwares which someone can only access by entering the correct password of your choice. now inorder to do so you have to follow some steps they are listed below.

(1) Select the desired folder that you want to hide. In my case I am selecting the folder named as example and I even stored a file named as secret inside the folder you can store the data that you want.

(2) Now, just make a new text file inside the folder. I am naming it "securedfolder.txt" . You can name it as per your choice.

(3) Now, just Click Here and you be redirected to a github repository in there you have to just copy the code from the repository and paste it into that text file that you just created in the last step.

(4) Now, just open the file in wich you pasted the code after opening that file move your cursor to line no.21 which is "if NOT %pass%== YOUR-PASSWORD goto FAIL" and replace the "YOUR-PASSWORD" with the password of your choice. In here I have used "examplepass" as my password for now. and save it.

(5) Now, just again open the file and click on the File option in the top left corner. Click on the Save as button and just change the extension the the file from "securedfolder.txt" to "securedfolder.bat" and save it. At last just delete the "txt" file.

(6) Now, just double click on the "securedfolder.bat" file and a folder named "Private" will be created automatically.

(7) Now, just move all the files and folders that you want to hide inside that "Private" folder that was just created in the last step.

(8) Now, just Again double click on the “LockedFolder.bat” A command prompt window will open asking you that “Are you sure you want to lock this folder?”. Type "Y" and press the enter key. the "private" folder will disappear.

(9) Now, To access your files again, double click on the "securedfolder.bat" file. A command prompt window will open asking you the password of the secured folder.

(10) Now, just type the password we created in the step no. 4 and press the enter key . After doing so you will be able to see the "Private" folder again.

(11) Now, when you will open the "Private" folder you will be able to access your files again.

(12) Now, inorder to hide your Private folder again, all you need to do is double-click on the “securedfolder.bat” file again and type "Y" and press the enter key. The "Private" folder will be disapperared again.
